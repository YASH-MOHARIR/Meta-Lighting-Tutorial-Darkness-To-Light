# Getting Started with Noesis Studio
Use this presentation to understand how to get started using Noesis Studio to make better User Interfaces in Horizon Worlds.

Included are instructions on how to use the `noesis2hzw` command line tool to translate Noesis Studio structures and data sets automatically into typescript code.
